Automatic playlist collector from YouTube subscriptions

This extension regularly adds new subscription videos to the "watch later" playlist

On this page, you can configure filters to ignore some videos

This is useful if you don't like:

  • live broadcasts of some users
  • very short or very long videos
  • specific video categories or themes
Start date for next collecting iteration

Videos published after this date will be added to the playlist in next iteration

Create conditions for ignoring the video.
These conditions will apply to each video together. If you need to apply conditions singly (as 'or'), create several filters.
